# IBM Data Science Professional Certificate

## Project Scenario and Overview
SpaceX is a leading Aerospace company making suborbtial spaceflights. SpaceX’s accomplishments include: sending spacecraft to the International Space Station, Starlink - a satellite internet constellation providing satellite Internet access,  sending manned missions to Space. One reason SpaceX can do this is the rocket launches are relatively inexpensive. 

SpaceX advertises Falcon 9 rocket launches on its website with a cost of 62 million dollars; other providers cost upwards of 165 million dollars each, **much of the savings is because SpaceX can reuse the first stage. Therefore, if we can determine if the first stage will land, we can determine the cost of a launch.** 

In this IBM's capstone project, I am undertaking the role of a data scientist working for a new rocket company, Space Y. The project's core obejective is to determine the price of each launch. This will be done by gathering information about Space X and creating dashboards and determine if SpaceX will reuse the first stage. Instead of using rocket science to determine if the first stage will land successfully, we are planning to train a machine learning model and use public information to predict if SpaceX will reuse the first stage.
